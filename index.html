Pour le développement d'API ou d'applications complètes, je sais m'appuyer sur ces valeurs sûres:
Le framework REACT pour un développement du frontend complet, puissant et rapide.
L'association NODE.js + Express pour une gestion du backend simple et fiable.
L'utilisation de bases de données relationnelles (MySQL) ou bien orientées documents (mongoDB).
